# 飞书先上线：你和我怎么配合

你负责登录账号和点确认，我负责告诉你每一步怎么填。

## 你先准备

- 飞书企业管理员账号，或有权限创建企业自建应用的账号。
- 一个测试群，例如 `仓库AI测试群`。
- n8n 公网地址。如果还没有，我们下一步先搭 n8n。
- OpenAI API Key。

## 第一次只做这些

1. 创建飞书企业自建应用。
2. 开启机器人。
3. 把机器人加入 `仓库AI测试群`。
4. 开启接收群消息权限。
5. 准备公网 n8n 地址。
6. 设置事件订阅地址到 n8n。
7. 让 n8n 收到飞书消息。
8. 让 OpenAI 分析消息。
9. 把重要消息先推送给你本人。
